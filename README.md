# Point of Sale (POS) Web Application

## 📌 Project Overview
This is a web-based Point of Sale (POS) system developed using PHP and MySQL.  
The system helps manage products, customers, and sales transactions efficiently.

## 🚀 Features
- Admin and Cashier Login System
- Product Management (Add, Edit, Delete)
- Customer Management
- Add to Cart & Checkout System
- Sales Report Generation
- Dashboard Overview

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap

## 📂 Installation Steps
1. Download the project
2. Import the database into phpMyAdmin
3. Update database connection in config.php
4. Run the project on XAMPP/WAMP server
